# Stora - Logistics Web App

مشروع ويب يستخدم Next.js لبناء موقع لشركة Stora للخدمات اللوجستية، يربط المحلات التجارية بعالم التجارة الإلكترونية ويوفر حلول لوجستية متكاملة.

## التقنيات المستخدمة

- [Next.js](https://nextjs.org) - إطار العمل الأساسي
- React
- Tailwind CSS
- TypeScript
- Git & GitHub
